# GitHub Actions Secrets Template – zoo-negara-landing

## Required Secrets

| Secret Name          | Description                                                                 | Example Value                |
|----------------------|-----------------------------------------------------------------------------|------------------------------|
| AWS_ACCESS_KEY_ID    | IAM user access key (with least privilege)                                  | AKIA...                      |
| AWS_SECRET_ACCESS_KEY| IAM user secret key                                                         | wJal...                      |
| AWS_DEFAULT_REGION   | AWS region for resources                                                    | ap-southeast-1               |
| SSH_PRIVATE_KEY      | Private SSH key (PEM) for EC2 access (generated by Terraform or provided)   | (PEM contents)               |
| S3_ARTIFACT_BUCKET   | S3 bucket for website artifacts deployment                                  | zoo-artifacts-unique123      |
| ALB_DNS_NAME         | ALB endpoint for health check (output from Terraform, can be set manually)  | my-alb-123.ap-southeast-1... |
| GRAFANA_ADMIN_PASS   | Grafana admin password (optional, set manually)                              | securepass123                |

## How to add secrets
- Go to repo "Settings" → "Secrets and variables" → "Actions"
- Add each secret with name and value.
